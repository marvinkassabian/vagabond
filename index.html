Controls

[W][A][S][D] to move player character [#]
Arrow keys [UP][LEFT][DOWN][RIGHT] to move screen
If the map spawns you in a wall, press [SPACE] to make a new map
If you want to switch between map types, press [CONTROL]
If you decide to switch map types, you will need to press [SPACE] again
Attack by clicking on a goblin that is adjacent to you

Game Information

You heal one hit point for every step
Goblins [%] only chase you when you're close to them
